\section{Supervised Learning}
Some basic terminologies:
\begin{itemize}
\item \textit{features}:  The set of attibutes, often represented as a vector, associated to an example.
\item \textit{Hypothesis space}: A set $\mathcal{F}$ of functions mapping features to the set of labels $\mathcal{Y}$.
\item \textit{Loss function}: A function $l$ that measures the difference, or loss, between a predicted label and a true label: $l:\mathcal{Y}Ã—\mathcal{Y}\to \mathbb{R}_{+}$, for example, $l(y,y')=(y-y')^{2}$.
\end{itemize}
